## 👀 Overview

📦 Frontend of the my Fullstack Tutorials project | [Tutorials Backend](https://github.com/ibrsec/tutortodo_app_backend)  </br></br>

🎯 <b>React Development:</b> Built a responsive frontend with React.js, offering a seamless user experience for managing tutorials.</br>

💾 <b>CRUD Operations:</b> Developed full CRUD functionality for efficiently managing tutorials and related data.</br>

🎨 <b>Styling:</b> Utilized Bootstrap to create a modern, responsive, and visually appealing user interface.</br>

<a name="quick-setup"></a>
## 🛫 Quick Setup

```sh
# clone the project
git clone https://github.com/ibrsec/tutortodo_app_frontend.git

# enter the project directory
cd tutortodo_app_frontend

# install dependency
npm install || yarn install

# develop
npm run dev || yarn start
```
